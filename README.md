# 🐍 Snake Game (Python)

A classic **Snake Game** built using **Python and Turtle Graphics**.
This project demonstrates basic game development concepts like movement logic, collision detection, score tracking, and file handling.

---

## 🎮 Features

* Keyboard controlled snake movement
* Grid-based gameplay
* Collision detection with walls and snake body
* Score tracking system
* High score saved using file handling
* Pause and restart functionality
* Executable version using **PyInstaller**

---

## 🛠 Technologies Used

* Python
* Turtle Graphics
* File Handling
* PyInstaller

---

## 🚀 How to Run the Game

1. Make sure **Python** is installed.
2. Download or clone this repository.

```bash
git clone https://github.com/yourusername/snake-game-python.git
```

3. Navigate to the project folder.

```bash
cd snake-game-python
```

4. Run the game.

```bash
python Snake.py
```

---

## 🎮 Controls

| Key   | Action          |
| ----- | --------------- |
| ⬆️    | Move Up         |
| ⬇️    | Move Down       |
| ⬅️    | Move Left       |
| ➡️    | Move Right      |
| SPACE | Start / Restart |
| P     | Pause           |

---

## 📦 Executable Version

The game can also be converted into a standalone executable:

```bash
pyinstaller --onefile --noconsole Snake.py
```

This will generate:

```
dist/Snake.exe
```
